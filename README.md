AT-CMD (Shell) Description & Revert Instructions

Description:

AT-CMD is a custom terminal/shell that replaces the standard Windows Explorer interface. It allows you to run commands, scripts, and manage your system in a command-line environment. Use it carefully, as it temporarily replaces the normal desktop interface.

How to Revert to Explorer:
